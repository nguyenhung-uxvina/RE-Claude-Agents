# Physical Effects Catalog

A reference catalog of physical effects for working principle identification, organized by category. Based on Pahl & Beitz systematic design methodology.

---

## How to Use This Catalog

1. **Identify the function** to be realized (from function structure)
2. **Determine the flow type** (energy, material, signal)
3. **Browse relevant category** below
4. **Match physical effect** to observed form design features
5. **Document in working principles catalog**

---

## 1. Mechanical Effects

### 1.1 Force and Motion

| Effect | Description | Typical Applications | Form Features |
|--------|-------------|---------------------|---------------|
| **Lever principle** | Force amplification/reduction via moment arm | Pliers, brakes, clutches | Pivot point, arms of different lengths |
| **Wedge principle** | Force direction change and amplification | Clamps, locks, chisels | Inclined surfaces, acute angles |
| **Screw principle** | Rotary to linear motion, mechanical advantage | Lead screws, jacks | Helical threads, pitch |
| **Cam mechanism** | Rotary to reciprocating motion | Valve trains, indexers | Cam profiles, followers |
| **Crank-slider** | Rotary to linear oscillation | Pistons, pumps | Crank, connecting rod, slider |
| **Gear transmission** | Speed/torque conversion, direction change | Gearboxes, differentials | Meshing teeth, pitch circles |
| **Belt/chain drive** | Remote power transmission | Conveyors, timing systems | Pulleys, belts, sprockets |
| **Friction** | Force transmission, braking | Clutches, brakes | Contact surfaces, friction materials |
| **Impact** | Momentum transfer | Hammers, pile drivers | Mass, velocity, contact surfaces |

### 1.2 Elastic Effects

| Effect | Description | Typical Applications | Form Features |
|--------|-------------|---------------------|---------------|
| **Spring force** | Energy storage, force generation | Suspensions, returns | Coils, leaves, torsion bars |
| **Elastic deformation** | Temporary shape change under load | Flexures, seals | Compliant geometry, elastomers |
| **Buckling** | Sudden collapse under compression | Snap-action switches | Slender columns, bistable elements |
| **Resonance** | Amplitude amplification at natural frequency | Filters, sensors | Tuned mass, stiffness |

### 1.3 Inertial Effects

| Effect | Description | Typical Applications | Form Features |
|--------|-------------|---------------------|---------------|
| **Centrifugal force** | Radial force from rotation | Governors, separators | Rotating masses, radial arms |
| **Gyroscopic effect** | Resistance to orientation change | Stabilizers, navigation | Spinning mass, gimbals |
| **Momentum conservation** | Reaction forces | Rockets, recoil systems | Mass ejection, buffers |

## 2. Electrical Effects

### 2.1 Electromagnetism

| Effect | Description | Typical Applications | Form Features |
|--------|-------------|---------------------|---------------|
| **Electromagnetic induction** | Voltage from changing magnetic flux | Motors, generators, transformers | Coils, magnetic cores, air gaps |
| **Lorentz force** | Force on current in magnetic field | Motors, actuators | Conductors, permanent magnets |
| **Magnetic attraction/repulsion** | Force between magnetic poles | Solenoids, relays, latches | Electromagnets, armatures |
| **Eddy currents** | Induced currents in conductors | Brakes, dampers, heating | Conductive masses, AC fields |
| **Magnetic saturation** | Limit of magnetization | Current limiting | Core materials, geometry |

### 2.2 Electrostatics

| Effect | Description | Typical Applications | Form Features |
|--------|-------------|---------------------|---------------|
| **Electrostatic attraction** | Force between charged bodies | Precipitators, coating | Electrodes, high voltage |
| **Capacitance** | Charge storage | Sensors, filters | Parallel plates, dielectrics |
| **Piezoelectric effect** | Voltage from mechanical stress | Sensors, actuators | Piezo crystals, ceramics |
| **Electrostriction** | Deformation from electric field | Precision actuators | Dielectric materials |

### 2.3 Resistance and Conductance

| Effect | Description | Typical Applications | Form Features |
|--------|-------------|---------------------|---------------|
| **Ohmic heating** | Heat from current flow | Heaters, fuses | Resistive elements |
| **Variable resistance** | Resistance change with position/condition | Potentiometers, strain gauges | Resistive tracks, grids |
| **Semiconduction** | Controlled conductivity | Transistors, diodes | Junctions, doped regions |
| **Superconductivity** | Zero resistance below critical temp | High-field magnets | Cryogenic materials |

## Quick Reference by Function Type

### Energy Conversion Functions

| From → To | Physical Effects |
|-----------|------------------|
| Electrical → Mechanical | Lorentz force, electromagnetic induction, piezoelectric |
| Mechanical → Electrical | Electromagnetic induction, piezoelectric, electrostatic |
| Thermal → Mechanical | Gas expansion, shape memory, bimetallic |
| Chemical → Electrical | Electrochemical reaction (batteries, fuel cells) |
| Light → Electrical | Photovoltaic, photoelectric |
| Mechanical → Thermal | Friction, compression |

### Force/Motion Functions

| Function | Physical Effects |
|----------|------------------|
| Amplify force | Lever, wedge, screw, hydraulic |
| Change direction | Gear, cam, linkage |
| Store energy | Spring, pneumatic, flywheel |
| Transmit power | Gear, belt, chain, shaft, hydraulic |
| Control motion | Friction, viscous damping, eddy current |

### Sensing Functions

| Measurand | Physical Effects |
|-----------|------------------|
| Position | Variable resistance, capacitance, inductance, optical |
| Force/Pressure | Piezoelectric, strain gauge, hydraulic |
| Temperature | Thermocouple, thermistor, bimetallic |
| Light | Photoelectric, photovoltaic |
| Motion | Doppler, electromagnetic induction |
